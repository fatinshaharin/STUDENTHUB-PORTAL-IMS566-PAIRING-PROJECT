# Project Overview
StudentHub Portal is an integrated academic management portal designed for university students to streamline access to academic information. This pairing project was developed as part of the IMS566 (Advanced Web Design Development and Content Management) course.

The portal provides a centralized platform for students to check university announcements, view classmate profiles, track academic grades, and manage their personal profiles.
